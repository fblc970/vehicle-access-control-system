# Vehicle Access Control System

## معرفی پروژه
این پروژه برای درس اصول طراحی نرم‌افزار طراحی شده است.

هدف این سیستم مدیریت ورود و خروج خودروها در سازمان‌ها، شرکت‌ها و مجتمع‌های مسکونی است.
